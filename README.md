# tweet-like

## Prerequisites
- PHP
- Symfony
- Docker

## How to run

- Check if it meets the requirements:
```
symfony check:requirements
```

- Clone the project:
```
git clone https://github.com/everton-araujo/tweet-like.git
```

- Install the dependencies:
```
composer install
```

- Run the project:
```
symfony server:start
```
